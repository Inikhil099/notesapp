

A modern, responsive note-taking application built with Next.js, featuring secure user authentication and full CRUD (Create, Read, Update, Delete) capabilities.


Features - 

User Authentication: Secure Sign Up and Login using JWT (JSON Web Tokens).

Secure Routing: Protected API routes  redirects for authenticated users only.

Full CRUD: Create, view, edit, and delete personal notes.

Database Integration: Persistent data storage using MongoDB and Mongoose.

Responsive UI: Clean, mobile-friendly design.

Feedback System: Toast notifications or alerts for successful actions and errors.
